Buzz is a simple number game where you count up and replace certain numbers with "buzz". You can play by yourself or against a bot.

Pick a difficulty: easy, medium, or hard.
Enter one or more buzz numbers (or type "random").
Choose if you want to play against a bot.
Start counting:
If the next number is divisible by or contains a buzz number, type "buzz".
Otherwise, type the number.
The game ends if you make a mistake.
